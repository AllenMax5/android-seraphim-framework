## 1. 产品概述

### 1.1 产品定位

一款面向小学教育的**跨学科素养评价数字化工具**，基于"四维·三阶·六翼"
评价模型，为教师、学生、家长及学校管理者提供全方位的素养评价解决方案。

### 1.2 目标用户

| 用户角色  | 核心需求           | 使用场景       |
|-------|----------------|------------|
| 教师    | 快速评价、批量录入、数据分析 | 课堂、活动、日常观察 |
| 学生    | 成长记录、作品展示、获得激励 | 课堂表现、活动参与  |
| 家长    | 了解孩子发展、配合家校共育  | 查看报告、参与活动  |
| 学校管理员 | 体系配置、数据管理、科研分析 | 后台管理、数据导出  |

### 1.3 核心价值

| 价值点   | 说明                       |
|-------|--------------------------|
| 科学评价  | 基于"四维·三阶·六翼"理论体系的权威评价模型  |
| 便捷记录  | 支持扫码即用，离线缓存，网络不佳也能记录评价数据 |
| 可视成长  | 雷达图、成长树等直观展示学生素养发展轨迹     |
| 家校联动  | 打通校、家、社、村多维场景，形成教育合力     |
| 个性化配置 | 六翼维度可自定义，适配不同学校的育人目标     |

## 2. 核心模型

### 2.1 四维·三阶·六翼 评价模型

#### 2.1.1 六翼（一级指标）

六翼代表六个核心素养维度，可根据学校特色进行配置：

| 维度 | 说明   | 示例            |
|----|------|---------------|
| 翼1 | 学科基础 | 语文、数学、英语等学科素养 |
| 翼2 | 艺术审美 | 音乐、美术、舞蹈等     |
| 翼3 | 科技创新 | 科学、编程、创客等     |
| 翼4 | 身心健康 | 体育、心理健康等      |
| 翼5 | 社会实践 | 劳动教育、志愿服务等    |
| 翼6 | 人文底蕴 | 阅读、传统文化等      |

**配置化特性**：学校可在后台修改六翼名称，适配本校育人目标。

#### 2.1.2 四级场景（评价场景）

- **L1 基础层**：日常课堂表现、作业完成情况
- **L2 活动层**：艺术节、益智节、阅读节、科技节、筑梦舞台等
- **L3 项目层**：寒暑假"1+1"任务、STEAM项目
- **L4 实践层**：校-家-社-村实践活动

#### 2.1.3 三阶进阶

- **第一阶**：基础达成
- **第二阶**：良好发展
- **第三阶**：卓越表现

#### 2.1.4 四维综合得分

将六翼具体表现映射到四个综合能力维度：

- 学习能力
- 实践能力
- 创新能力
- 合作能力

## 3. 功能需求

### 3.1 教师端

#### 3.1.1 扫码/选人评价（P0 — 核心）

**功能描述**：在活动场景中快速评价学生

**核心流程**：

1. 选择评价场景（如"艺术节评价"、"阅读节评价"）
2. 选择班级
3. 选择学生（扫码或列表选择）
4. 选择对应六翼指标
5. 一键打分（支持三级进阶）

**界面元素**：

- 扫码入口（支持学生二维码扫描）
- 班级/学生选择器
- 六翼指标卡片（彩色区分）
- 三级进阶评分（⭐/⭐⭐/⭐⭐⭐）
- 快捷评语标签
- 语音评价输入

#### 3.1.2 批量评价（P1）

**功能描述**：全班跳绳测试、课堂表现等场景批量录入

**核心流程**：

1. 选择评价活动
2. 批量选择学生（全班/多选）
3. 统一或分别评分
4. 批量提交

**界面元素**：

- 学生列表（支持多选/全选）
- 批量评分面板
- 快速录入模式
- 导入模板（Excel）

#### 3.1.3 活动任务发布（P1）

**功能描述**：发布寒暑假"1+1"任务、STEAM项目等

**核心流程**：

1. 创建任务（标题、描述、要求、截止时间）
2. 选择目标班级/学生
3. 设置评价指标关联
4. 发布任务
5. 学生上传作品后线上评价

#### 3.1.4 班级报告（P1）

**功能描述**：一键生成班级四维雷达图，快速定位班级整体优劣势

**功能特性**：

- 自动生成班级素养雷达图
- 六翼均分对比
- 班级优势/待提升维度分析
- 支持导出PDF报告

#### 3.1.5 评语库（P2）

**功能描述**：预设针对不同六翼表现的评语模板，提高效率

**功能特性**：

- 按六翼分类的评语模板
- 智能推荐评语（基于评分）
- 自定义评语收藏
- 常用语快捷输入

#### 3.1.6 数据看板（P2）

**功能描述**：查看班级六翼均分趋势图，追踪阶段性成长

**功能特性**：

- 时间维度筛选（周/月/学期/学年）
- 六翼发展趋势曲线
- 与年级平均对比
- 异常数据预警

---

### 3.2 家长/学生端

#### 3.2.1 个人成长树/星球（P0 — 核心）

**功能描述**：游戏化激励机制，每次获得评价成长树长出新叶或获得星星

**功能特性**：

- 3D/2D可视化成长树/星球
- 不同六翼对应不同颜色/形态
- 获得评价时的动画反馈
- 成就徽章系统
- 等级进阶（幼苗→小树→大树→参天大树）

#### 3.2.2 雷达图报告（P0 — 核心）

**功能描述**：随时查看个人当学期、学年、整个小学阶段的动态雷达图

**功能特性**：

- 四维雷达图展示
- 时间轴切换（学期/学年/学段）
- 历史对比（与上学期对比）
- 雷达图动画演进
- 分享功能（生成精美卡片）

#### 3.2.3 电子档案袋（P1）

**功能描述**：上传作品，申请探究层加分

**支持作品类型**：

- 绘画作品（图片）
- 作文/文章（文字+图片）
- 演奏视频（视频）
- 环保时装秀照片（图片）
- 其他创意作品

**核心流程**：

1. 选择作品类型
2. 上传作品（支持多图/视频）
3. 填写作品说明
4. 选择关联六翼维度
5. 提交审核
6. 教师审核后获得加分

#### 3.2.4 实践活动打卡（P1）

**功能描述**：参与"校-家-社-村"活动，拍照打卡获取积分

**支持活动类型**：

- 社区环保活动
- 家家亲活动
- 志愿服务
- 传统文化体验
- 其他实践活动

**核心流程**：

1. 查看活动列表
2. 参与活动
3. 拍照/视频打卡
4. 填写活动心得
5. 获得积分/评价

#### 3.2.5 成长轨迹（P2）

**功能描述**：查看1-6年级六翼素养变化曲线

**功能特性**：

- 长期趋势曲线图
- 关键节点标记（获奖、重大突破）
- 学年对比
- 毕业总览报告

#### 3.2.6 家校建议（P2）

**功能描述**：接收教师推送的个性化成长建议和家庭实践任务

**功能特性**：

- 个性化建议推送
- 家庭实践任务
- 亲子活动推荐
- 反馈确认机制

## 5. 核心业务流程

### 5.1 日常评价流程

```
教师选择场景 → 选择班级/学生 → 六翼指标评分 → 添加评语 → 提交
                                    ↓
                              学生/家长实时收到通知
                                    ↓
                              成长树更新 + 雷达图刷新
```

### 5.2 活动评价流程

```
管理员创建活动 → 配置评价指标 → 发布活动
                                    ↓
                              教师参与活动评价
                                    ↓
                              学生获得活动积分/徽章
```

### 5.3 作品提交流程

```
学生上传作品 → 填写作品信息 → 选择六翼维度 → 提交审核
                                                  ↓
                                           教师审核评分
                                                  ↓
                                           获得探究层加分
```

### 5.4 实践活动流程

```
家长/学生查看活动 → 参与活动 → 拍照打卡 → 提交心得
                                             ↓
                                      教师确认/评分
                                             ↓
                                      获得实践积分
```

## 7. 界面设计规范

### 7.1 设计原则

- **儿童友好**：色彩明快，图标可爱，符合小学生审美
- **简洁高效**：教师端注重效率，操作路径短
- **数据可视化**：大量运用图表、雷达图、趋势曲线

### 7.2 色彩体系

| 用途      | 色值         |
|---------|------------|
| 主色（教育蓝） | #4A90D9    |
| 六翼-学科基础 | #FF6B6B（红） |
| 六翼-艺术审美 | #FF9F43（橙） |
| 六翼-科技创新 | #F9CA24（黄） |
| 六翼-身心健康 | #6DD55A（绿） |
| 六翼-社会实践 | #22D3EE（青） |
| 六翼-人文底蕴 | #A855F7（紫） |
| 成功      | #52C41A    |
| 警告      | #FAAD14    |
| 错误      | #F5222D    |

### 7.3 字体规范

- **中文**：PingFang SC / Noto Sans SC
- **标题**：18-20sp，加粗
- **正文**：14-16sp，常规
- **辅助**：12sp，次要信息

## 8. 非功能需求

### 8.1 性能要求

| 指标    | 要求           |
|-------|--------------|
| 启动时间  | ≤ 2s         |
| 页面切换  | ≤ 300ms      |
| 评价提交  | ≤ 1s         |
| 雷达图渲染 | ≤ 500ms      |
| 支持并发  | 1000+ 用户同时在线 |

### 8.2 离线支持

- 评价数据本地队列，联网后自动同步
- 图片压缩后暂存，联网后上传
- 离线状态下可查看历史数据

### 8.3 数据安全

- 学生隐私数据加密存储（AES-256）
- 敏感操作权限控制
- 数据传输使用HTTPS
- 定期数据备份机制

### 8.4 兼容性

- Android 8.0 (API 26) 及以上
- 适配手机、平板多种屏幕尺寸
