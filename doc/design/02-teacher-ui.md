# 👨‍🏫 教师端界面设计

> **文档版本**: v1.0.0  
> **最后更新**: 2026-04-09  
> **目标用户**: 小学教师

---

## 1. 设计目标

### 1.1 核心目标

| 目标     | 说明         | 衡量标准      |
|--------|------------|-----------|
| **高效** | 单次评价 ≤ 30秒 | 操作步骤 ≤ 5步 |
| **便捷** | 随时随地可评价    | 支持离线、扫码   |
| **清晰** | 信息一目了然     | 减少认知负担    |
| **可靠** | 数据不丢失      | 自动同步、备份   |

### 1.2 设计策略

- **一键优先**: 最常用的功能一步可达
- **默认智能**: 智能填充默认值，减少输入
- **批量操作**: 支持批量评价，提高效率
- **离线可用**: 网络不佳时也能正常使用

## 6. 动效规范

### 6.1 评价提交成功

```
序列:
1. 按钮显示加载动画 (300ms)
2. 全屏显示成功动画:
   - 成长树生长动画 (600ms)
   - 星星粒子爆炸 (400ms)
   - "评价成功"文字淡入 (200ms)
3. 自动返回首页 (500ms delay)
```

### 6.2 雷达图加载

```
序列:
1. 网格线从内向外绘制 (400ms)
2. 数据轴依次出现 (50ms each)
3. 数据点从中心向外移动 (600ms)
4. 填充区域渐变显示 (300ms)
```
