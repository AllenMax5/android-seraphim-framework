# 👦 学生/家长端界面设计

> **文档版本**: v1.0.0  
> **最后更新**: 2026-04-09  
> **目标用户**: 小学生及其家长

---

## 1. 设计目标

### 1.1 核心目标

| 目标       | 说明            | 衡量标准         |
|----------|---------------|--------------|
| **激励**   | 让成长可视化，激发内在动力 | 成长树活跃率 > 80% |
| **易懂**   | 小学生能独立使用      | 7岁孩子可无指导操作   |
| **成就感**  | 每一点进步都被看见     | 及时的正向反馈      |
| **家校连接** | 家长了解孩子在校表现    | 家长日活 > 60%   |

### 1.2 设计策略

- **游戏化**: 成长树、徽章、等级系统
- **即时反馈**: 评价获得时立即视觉反馈
- **趣味化**: 可爱插画、动画、音效
- **家长视角**: 增加数据解读和成长建议

- 总时长: 800ms

## 5. 儿童友好设计细节

### 5.1 语言风格

| 场景   | 成人版         | 儿童版            |
|------|-------------|----------------|
| 获得评价 | "获得科技创新评价"  | "你的创意被看到啦！"    |
| 升级提示 | "等级提升至Lv.3" | "哇！你的小树长大啦！"   |
| 鼓励语  | "继续加油"      | "你是最棒的！继续发光吧！" |
| 完成任务 | "任务完成"      | "太棒了！又收获一颗星星！" |

### 5.2 音效设计

| 场景    | 音效风格        |
|-------|-------------|
| 获得评价  | 欢快的上升音 + 铃铛 |
| 升级    | 庆祝音效 + 烟花声  |
| 点击按钮  | 轻快的"啪"声     |
| 错误提示  | 温和的"咚"声     |
| 成长树互动 | 风吹树叶声       |

### 5.3 手势操作

| 手势    | 操作     | 反馈   |
|-------|--------|------|
| 点击成长树 | 互动动画   | 树叶晃动 |
| 长按成长树 | 惊喜动画   | 掉落果实 |
| 下拉刷新  | 成长树浇水  | 水滴动画 |
| 左右滑动  | 切换时间维度 | 页面滑动 |
| 双击徽章  | 放大查看   | 徽章翻转 |
