## 1. 设计原则

### 1.1 核心设计理念

| 原则        | 说明                    | 设计体现          |
|-----------|-----------------------|---------------|
| **成长导向**  | 每个孩子都是独特的种子，需要被看见、被鼓励 | 视觉反馈强调进步而非比较  |
| **温暖专业**  | 教育场景需要信任感，同时不失温度      | 圆润的边角、柔和的色彩过渡 |
| **简单高效**  | 教师时间宝贵，操作必须直观快捷       | 一键操作、清晰的视觉层级  |
| **游戏化激励** | 通过视觉奖励激发学生内在动力        | 成长树、徽章、动画反馈   |

### 1.2 设计关键词

```
🌱 温暖    🎨 多彩    ⚡ 高效    🌳 成长    ✨ 惊喜
```

## 9. 动效规范

### 9.1 动画原则

- **目的性**: 每个动画都有明确目的（反馈、引导、愉悦）
- **快速**: 动画时长200-400ms，不拖沓
- **自然**: 使用 ease-in-out 或 spring 曲线
- **克制**: 不过度使用动画，避免干扰

### 9.2 动画时长

| 类型      | 时长      | 用途        |
|---------|---------|-----------|
| **微交互** | 150ms   | 按钮点击、图标切换 |
| **标准**  | 300ms   | 页面切换、弹窗   |
| **复杂**  | 400ms   | 成长动画、庆祝效果 |
| **持续**  | 1000ms+ | 加载、引导动画   |

### 9.3 缓动函数

```css
/* 标准 */
--ease-default: cubic-bezier(0.4, 0, 0.2, 1);

/* 减速 */
--ease-decelerate: cubic-bezier(0, 0, 0.2, 1);

/* 弹性 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);

/* 线性 */
--ease-linear: linear;
```

### 9.4 关键动画

#### 页面转场

```
进入: 从右滑入 + 淡入, 300ms, ease-decelerate
退出: 向左滑出 + 淡出, 250ms, ease-default
```

#### 卡片悬浮

```
触发: Hover / Touch
效果: Y轴上移4dp + 阴影加深
时长: 200ms
缓动: ease-spring
```

#### 成长树动画

```
生长动画:
- 从种子到幼苗: 缩放 + 旋转, 600ms
- 新叶生长: 路径动画, 400ms each
- 闪光效果: 400ms

获得评价时:
- 树叶飘落: 物理模拟
- 星星弹出: scale 0→1.2→1, 300ms
```

#### 雷达图动画

```
绘制动画:
- 从中心向外扩展
- 每轴依次绘制
- 总时长: 800ms
```

#### 评分反馈

```
星星点击:
- Scale: 1 → 1.3 → 1
- 颜色: 灰 → 黄
- 时长: 200ms
- 缓动: ease-spring

连击效果:
- 星星数量越多，动画越大
- 配合粒子效果
```

## 11. 无障碍设计

### 11.1 色彩对比度

- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1
- 交互元素对比度 ≥ 3:1

### 11.2 触摸目标

- 最小触摸目标: 48dp × 48dp
- 相邻目标间距: ≥ 8dp

### 11.3 文字缩放

- 支持系统字体大小设置 (100%-200%)
- 布局自适应文字大小
